# Project Rules

## Git Safety
- NEVER commit, push, or run any git commands without explicit user permission
- All changes must stay local until the user says "push" or "deploy"
- Always ask before any destructive operations

## Project Info
- Python/Flask inspection web app
- Live URL: https://inspection-web.onrender.com/
- Local URL: http://localhost:5000
- Bilingual: Arabic and English
- Database: PostgreSQL
